# 🌍 Un Mondo Migliore

> *«C'è bisogno soprattutto d'uno slancio generoso, fosse anche un sogno matto.»*
> — Guccini, *Don Chisciotte*

Un sogno forse pazzerello, portato avanti scherzando sul serio: costruire un
mondo migliore come fosse un **social network reale**, vissuto nella vita di
tutti i giorni — finché il sogno non diventa realtà.

L'idea di fondo è semplice: **ognuno contribuisce con quello che gli piace
fare.** Nel mondo migliore ciascuno dà il meglio di sé, perché fa ciò che ama.
Nessun obbligo — ma è bello dare anche poco, ciò che sai fare meglio.

È nato in una **realtà parrocchiale**, ma non resta dentro quel confine: qui
ogni credo e ogni diversità è una ricchezza reale, nessuna idea prevale su
quella di un altro, e quando serve scegliere insieme **ogni persona pesa
uguale**.

Si comincia da dove si è. Da qualche parte bisogna pure iniziare: noi iniziamo
da qui.

## Vedere il sito

🌍 **Online:** https://manzolo.github.io/UnMondoMigliore/

Il sito è pubblicato con GitHub Pages dalla cartella [`docs/`](docs/), che viene
generata dal comando `make`. Per aggiornarlo online: `make` e poi un commit di
`docs/`.

Lo si può anche generare e vedere in locale (serve solo `python3`, nessuna
dipendenza):

```sh
make          # genera il sito in dist/
make open     # genera e apre nel browser
make serve    # genera e avvia un server su http://localhost:8000
make clean    # rimuove dist/
make help     # elenco dei comandi
```

## Come si partecipa

1. Leggi il [manifesto](MANIFESTO.md).
2. Pensa a **una cosa** che ti piace fare bene.
3. Falla per qualcun altro.
4. Se ti va, scrivilo nel [diario](diario/).

Tutto qui. Il resto cresce da solo.
